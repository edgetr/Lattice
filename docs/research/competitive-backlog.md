# Prioritized competitive backlog

Research basis and exact upstream SHAs are recorded in [session-outbox-fast-switch.md](session-outbox-fast-switch.md). These are scoped backlog items, not claims of current Lattice capability and not part of the outbox implementation.

1. **P1 — Integrated terminal and file tree.** Preserve terminals at workspace scope across chat switches; add a lazy file tree with single-click preview and explicit pinned-open behavior. Source evidence: T3 [`ThreadTerminalDrawer.tsx`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/web/src/components/ThreadTerminalDrawer.tsx) and [`FileBrowserPanel.tsx`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/web/src/components/files/FileBrowserPanel.tsx); OpenCode [terminal switch regression](https://github.com/anomalyco/opencode/blob/571e7b852f82415faf65466e1536357a048bdf5a/packages/app/e2e/regression/terminal-tab-switch.spec.ts#L19-L45) and [file-browser switch regression](https://github.com/anomalyco/opencode/blob/571e7b852f82415faf65466e1536357a048bdf5a/packages/app/e2e/regression/file-browser-sidebar-tab-switch.spec.ts#L17-L61).
2. **P1 — Review UX.** Provide a per-session changed-file list, lazy diffs, stable path-keyed previews, and inline comments that enter composer context only after explicit user action. Source evidence: T3 [`review.ts`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/web/src/state/review.ts); OpenCode [`review-panel-v2.tsx`](https://github.com/anomalyco/opencode/blob/571e7b852f82415faf65466e1536357a048bdf5a/packages/app/src/pages/session/v2/review-panel-v2.tsx#L41-L164).
3. **P2 — Preview picker.** Add an explicit preview target/device picker and reviewable annotations, with no implicit external navigation. Source evidence: T3 [`PickPreload.ts`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/desktop/src/preview/PickPreload.ts) and [`PreviewPanel.tsx`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/web/src/components/preview/PreviewPanel.tsx).
4. **P2 — Mobile/remote continuity.** Design an authenticated environment identity, sequence-aware replay, scoped permissions, and durable outbox receipts before adding remote execution. Treat this as a security architecture project, not a transport toggle. Source evidence: T3 [remote architecture](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/docs/architecture/remote.md); OpenCode [remote session isolation regression](https://github.com/anomalyco/opencode/blob/571e7b852f82415faf65466e1536357a048bdf5a/packages/app/e2e/regression/remote-session-settings.spec.ts#L13-L81).
5. **P2 — Context breakdown.** Extend Lattice's honest local estimate into system/user/assistant/tool/other categories, show provider-reported totals separately, and label reconciliation as estimated. Source evidence: OpenCode [`session-context-breakdown.ts`](https://github.com/anomalyco/opencode/blob/571e7b852f82415faf65466e1536357a048bdf5a/packages/app/src/components/session/session-context-breakdown.ts#L3-L131). T3 currently supplies a total usage meter rather than this category breakdown: [`ContextWindowMeter.tsx`](https://github.com/pingdotgg/t3code/blob/3513fa04fbf12c1d4fa2b8d07cfc7f0905714d31/apps/web/src/components/chat/ContextWindowMeter.tsx).
